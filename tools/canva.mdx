# Canva

Connect your Canva account and create designs through Mira.

## Connect

```
Connect my Canva
```

## Use cases

### Create designs

```
Create a Canva presentation about our Q4 results
Design an Instagram story for the product launch
Make a social media post about the new feature
```

### Export

```
Export the last design as PNG
Download my Canva presentation as PDF
```

<Tip>Be specific about the type of design — *"Instagram story"*, *"A4 poster"*, *"16:9 presentation"* — for best results.</Tip>
