# Gmail

Connect your Gmail account and manage emails without leaving Telegram.

## Connect

```
Connect my Gmail
```

Mira sends a secure OAuth link. Click, authorize, done.

## Use cases

### Read emails

```
Check my unread emails
Show me the latest email from Sarah
Summarize the last 10 emails in my inbox
```

### Send emails

```
Send an email to john@company.com:
Subject: Meeting tomorrow
Body: Hi John, confirming our meeting at 3pm tomorrow. See you there!
```

### Search & filter

```
Find emails about the Q4 report
Show me unread emails from this week
Search for emails with attachments from Alex
```

### Archive & organize

```
Archive all emails from newsletters
Mark the email from boss as important
```

<Tip>Mira can summarize long email threads — just ask: *"Summarize this email thread"*.</Tip>
