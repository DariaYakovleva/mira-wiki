# Figma

Connect your Figma account and access design files through Mira.

## Connect

```
Connect my Figma
```

## Use cases

### Browse files

```
List my Figma projects
Show me files in the "Design System" project
Get details about the file "Mobile App v2"
```

### Get information

```
What components are in the design system?
Show me the latest version of the homepage design
```

<Tip>Figma integration is read-only — Mira can view and describe designs, but cannot edit them.</Tip>
