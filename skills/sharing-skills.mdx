# Sharing Skills

Created a great skill? Share it with anyone via a simple Telegram link.

## How to share

```
Share my "Content Writer" skill
```

Mira generates a unique `t.me` link. When someone opens it:

1. They see a preview of the skill — name, description, category
2. They can tap **Install** to add it to their Mira
3. The skill starts working immediately

## Publishing

When you share a skill, Mira asks if you want to:

- **Publish to community catalog** — visible to all Mira users browsing skills
- **Keep link-only** — only people with the link can install it

Choose based on how widely you want to share.

## Managing shared skills

```
List my skills
Make the "Content Writer" skill public
Make the "Sprint Bot" skill private
```

## What users see

When someone opens your skill link:

- **Skill name** and **description**
- **Category** (dev, design, productivity, etc.)
- **Install** button
- They do NOT see your full instructions — those remain private

## Tips

<Tip>Share skills with your team to standardize workflows — everyone gets the same behavior from Mira.</Tip>

<Tip>The best shared skills solve a specific, recurring problem. Think: "What do I ask Mira to do every week?"</Tip>

<Info>
  System skills (built-in) cannot be shared. Only user-created skills can be shared.
</Info>
