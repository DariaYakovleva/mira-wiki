# API Reference

Mira offers APIs for developers who want to integrate AI capabilities into their own applications, bots, and services.

## Available APIs

### Generation API

Generate images, videos, and music programmatically via HTTP endpoints.

- **Image generation** — create visuals from text descriptions
- **Image editing** — transform and restyle existing images
- **Video generation** — produce short video clips from prompts
- **Music generation** — compose original music and songs

### Skills API

Create, manage, and execute custom skills programmatically.

- **Create skills** — define behavioral scenarios via API
- **Execute skills** — trigger skills on demand
- **List & manage** — enumerate, update, and delete skills
- **Share skills** — generate shareable links

## Use cases

- Build custom Telegram bots powered by Mira
- Embed Mira's AI into your product
- Automate content generation workflows
- Create agent-to-agent communication protocols
