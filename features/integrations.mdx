## Overview

Mira connects to your favorite tools via [Composio](https://composio.dev) — a platform that supports 500+ integrations. Once connected, Mira can take real actions: send emails, create calendar events, push code to GitHub, update Notion pages, and much more.

## How to connect

<Steps>
  <Step title="Ask Mira to connect">
    Just say: *"Connect my Gmail"* or *"Connect GitHub"*
  </Step>
  <Step title="Click the auth link">
    Mira sends you a secure OAuth link. Click it and authorize in your browser.
  </Step>
  <Step title="Start using it">
    That's it! Mira can now act on your behalf.
  </Step>
</Steps>

## Example commands

```
Check my emails from today
Create a meeting tomorrow at 3pm with John
Create a GitHub issue: "Fix login bug"
Add a row to my Google Sheet with today's sales data
Post a message to #general in Slack
```

## Automations & Triggers

Mira can also **watch for events** and notify you automatically:

```
Notify me when I get a new GitHub issue
Alert me when a new email arrives from my boss
```

<Note>
  Triggers run in the background and send you a Telegram message when the event fires.
  Maximum 10 active triggers per user.
</Note>
