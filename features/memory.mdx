## How memory works

Mira stores two types of information about you:

### Profile
Basic facts that help Mira personalize responses:
- Name, age, location
- Occupation
- Interests and hobbies
- Preferred language
- Timezone
- Communication style

### Facts
Specific things you share during conversations:
- *"I prefer bullet points over long paragraphs"*
- *"I'm learning Spanish"*
- *"I don't like formal language"*
- *"My startup is in the fintech space"*

## Visibility

| Context | Visibility |
|---------|------------|
| Private chat | Facts are private by default |
| Group chat | Facts are public (visible to group members) |

## How to update your memory

Just tell Mira naturally:

```
Remember that I prefer short answers
My name is Dasha
I'm based in London
I work as a Product Manager
```

Mira will save it automatically.

## Communication style

You can customize how Mira talks to you:

- **Friendly** — warm and casual
- **Listener** — empathetic and supportive  
- **Robot** — precise and technical
- **Nerd** — deep dives and details
- **Cynic** — dry humor, no fluff

Just say: *"Be more casual with me"* or *"I prefer technical language"*.

<Tip>
  You can also set your communication style in the Mira mini-app.
</Tip>
