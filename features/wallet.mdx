## Overview

Every Mira user gets a personal **TON wallet** — created automatically on first use. No seed phrases to write down, no separate apps.

<Warning>
  Mira's wallet currently runs on **testnet**. Funds have no real monetary value.
  Top up your test wallet via [@testgiver_ton_bot](https://t.me/testgiver_ton_bot).
</Warning>

## Example commands

```
What's my wallet balance?
What's my wallet address?
Send 1 TON to @username
Send 5 USDT to 0:abc...def
Swap 2 TON to USDT
Show my recent transactions
```

## Getting your address

Ask Mira: *"What's my wallet address?"* — she'll show it in a copyable format.

## Transaction history

Ask Mira: *"Show my recent transactions"* — she'll list incoming and outgoing transfers with amounts and timestamps.
