## Generate images

Describe what you want and Mira will generate it. Costs **30 tokens** per image.

```
Generate a portrait of a woman in Art Nouveau style, floral background
```

### Aspect ratios

You can specify the aspect ratio:

| Ratio | Best for |
|-------|----------|
| `9:16` | Stories, vertical content (default) |
| `16:9` | Landscapes, presentations |
| `1:1` | Profile pictures, posts |
| `4:5` | Instagram feed |
| `3:2` | Photography style |

## Edit images

Send any photo to Mira and ask her to transform it. Costs **30 tokens**.

**Examples:**
- *"Make this photo anime style"*
- *"Add a forest background"*
- *"Change the color palette to warm tones"*
- *"Make it look like a painting"*

<Tip>You can send multiple images and ask Mira to combine or compare them.</Tip>

## Analyze images

Send a photo and ask Mira what's in it — **free of charge**.

```
What's in this photo?
Read the text on this sign.
Describe the mood of this scene.
```
